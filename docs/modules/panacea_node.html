untildify

  • untildify(path: string): string
    • Defined in node/src/string/untildify.ts:20

    将波浪号路径转换为绝对路径

    remarks

    波浪号路径是指以 ~ 开头的路径，例如 ~/foo/bar。

    example
    const path = untildify('~/foo/bar');
    console.log(path); // /home/username/foo/bar
    since

    0.1.0

    Parameters

    • path: string

      波浪号路径

    Returns string

    转换后的绝对路径
